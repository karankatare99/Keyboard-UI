# UI Keyboard

A keyboard UI component built with Next.js, React and Tailwind CSS.

Live demo (deployed): **https://keyboard-chkg2uwwe-ricks-projects-29cad068.vercel.app/**

## Tech stack
- Next.js
- React
- TypeScript
- Tailwind CSS (with PostCSS)
- ESLint
- Prettier
- Vercel (deployment)

## Quick start

Install deps and run the dev server:

```bash
npm install
npm run dev
# or
pnpm install
pnpm dev
```

Open http://localhost:3000.

## Key files
- app/page.tsx — main page
- components/keyboard.tsx — Keyboard component
- components/ButtonStyles.tsx — key/button components and styles
- components/svgs.tsx — key SVGs and symbols
- package.json, tsconfig.json, tailwind.config.js — project config
